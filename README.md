# quollr: An R Package for Visualizing 2-D Models from Non-linear Dimension Reductions in High Dimensional Space

This repo contains the materials to fully reproduce the paper titled "quollr: An R Package for Visualizing 2-D Models from Non-linear Dimension Reductions in High Dimensional Space".

## Description

- [`data`](data): The data sets used in the empirical application.
- [`figures`](figures): Figures used in the paper.
- [`scripts`](scripts): R scripts to reproduce the results presented in the paper.
- The manuscript is written using [Rmarkdown](https://rmarkdown.rstudio.com/). The TeX and PDF files for the paper are generated by rendering the [`paper-quollr.rmd`](paper-quollr.rmd) file.
